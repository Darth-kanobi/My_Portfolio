Intro

"Whether you want to uncover the secrets of the universe or you just want to pursue a career in the 21st century, basic computer programming is an essential skill to learn"
-------Stephen Hawking-------

To program is to understand what happens what behind the curtain. It was this very curiosity that got me into coding as a resetless 16 year old and I haven't looked back since!

Like most people, my first introduction to programming was BASIC. A simple language, which taught me the fundamentals and I also understood how syntax and functions works. I even went as far as to code a simple ping-pong game in BASIC, which was really fun to do.

In due course of my schoolwork, I was introduced to the basics of html, css and even a bit of scratch. But it wasn't until 11th grade that I would really get into coding. My school curriculum required me to learn java, and thats what I did. I started exploring OOP concepts and fully immersed myself into the process of coding and programming.

It was here that I realised that coding was not excatly about which language you used or which code editor you used, it was about the rewiring the way our brains work. By changing the way we approach coding problems, we can appreciate the process of programming more.

Java quickly became my favorite language to code on. The OOPs concepts made it really easy to write code and understand I really liked the various libraries that java offered, and the string formmating was also really easy to work with. So I started to go beyond the scope of my textbooks and took a closer look into the use cases of Java.

And to my surprise, Java turned out to be a allrounder! We can do pretty much anything with Java, but the main use cases of Java was app development and web development and Java GUI was also used extensive.Here are some of the awesome projects that I have done with different programming languages.

So I got into it. I started to program with what little time I got. Made more and more mistakes, which really slowed down my progress. This is also where I started to understand the difference between coding and programming.

Coding is just sloving math and pattern problems by using commands issued to the computer. Programming on the other hand is applying what we learnt while coding to solve real world problems. Because that is the endgame of learning to program, automating certain tasks for the benefit of humanity!

Project

I have created a python program as part of the class 12th project for Computer science.

FOOD BOOKING SYSTEM

The python project on Food Booking System was to manage the details of booking, menu details, payments, and order.

The project is totally built at administrative end and only administrator is guaranteed the access to the software.

The purpose of the project was to build an application program to reduce the manual work for managing the booking, discounts, and payments.

It tracks all the details about menu, discount, and payments; it also prints various reports as per input given by the user.

The program ran on python and was connected to an SQL database where the details of the customers, the employees working at the hotel or restaurant and the billing details that took place were stored.
